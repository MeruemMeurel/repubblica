# Giornale Quotidiano - Progetto PPM

Progetto realizzato per il corso di Progettazione e Produzione Multimediale.

## Tecnologie utilizzate

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media query per il responsive design
- Nessun framework CSS
- Nessun JavaScript

## Struttura del progetto

```text
/
├── index.html
├── css/
│   ├── style.css
│   └── responsive.css
└── README.md
```

## Struttura della pagina

La homepage contiene:

- topbar con menu, ricerca, notifiche, accesso e abbonamento;
- testata del quotidiano;
- navigazione principale;
- spazio pubblicitario;
- risultati sportivi;
- sezione sportiva in evidenza;
- sezione “Primo Piano”;
- sidebar con video, newsletter, podcast e rubriche;
- feed di notizie;
- sezione economia;
- footer informativo.

Il CSS è stato diviso in due file per separare gli stili principali dalle regole responsive.

## Responsive design

Il sito è responsive e si adatta a desktop, tablet e smartphone.

Su schermi piccoli:

- il layout passa da due colonne a una colonna;
- le griglie di articoli vengono semplificate;
- la navigazione diventa scrollabile orizzontalmente;
- sidebar e contenuto principale vengono mostrati in sequenza.

## Accessibilità

Il sito è stato controllato con WAVE.

Risultato finale:

- Errors: 0
- Contrast Errors: 0

Sono presenti alcuni warning dovuti all’uso di immagini placeholder con testi alternativi simili o ripetuti.
